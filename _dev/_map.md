## Key routes (vercel.json)

```
/api/services/status  ->  /api/status        (cloud model ping)
/api/health           ->  /api/health         (health + Supabase check)
/api/data             ->  /api/data           (conversations, history)
/ask                  ->  /api/query/[action]?action=ask  (main AI endpoint)
/agent                ->  /api/agent          (GitHub operations)
```

## Workflow: Using Coder on a project

```
1. Open Coder: https://mobius.vercel.app
2. Project: Open -- navigate to project folder (e.g. GPRTool)
3. Project: List  -- confirm correct folder
4. Project: Read [filename] -- load a file into context
5. Code: File [filename] [instruction] -- read + send in one step
6. Code: Fix / Code: Explain / Code: Review -- work with loaded file
7. Debug: [error message] -- start debug pipeline for a known error
8. Ask: [model] [question] -- direct AI query for any question
```
