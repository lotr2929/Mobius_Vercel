Here's the exact `updateNorthRotation` function from `app/js/north-point-2d.js` that applies CSS transforms to the north indicators:

```javascript
function updateNorthRotation() {
    // Apply rotation to 2D north needle
    tnNeedleEl.style.transform = `rotate(${designNorthAngle}deg)`;

    // Apply rotation to 3D north group
    dnGroupEl.style.transform = `rotateZ(${designNorthAngle}deg)`;
}
```

This function:
1. Takes the current `designNorthAngle` value
2. Applies a CSS rotation to the 2D north needle element (`tnNeedleEl`)
3. Applies the same rotation to the 3D north group element (`dnGroupEl`) using `rotateZ`

The rotation is applied in degrees and affects both the 2D and 3D north indicators simultaneously.
