### Codestral
Here's the exact `updateNorthRotation` function from `app/js/north-point-2d.js` and `app/js/north-point-3d.js` that applies CSS transforms to the north indicators:

```javascript
// In app/js/north-point-2d.js
function updateNorthRotation() {
    const angle = designNorthAngle;
    tnNeedleEl.style.transform = `rotate(${angle}deg)`;
}

// In app/js/north-point-3d.js
function updateNorthRotation() {
    const angle = designNorthAngle;
    dnGroupEl.style.transform = `rotateY(${angle}deg)`;
}
```

These functions:
1. Take the current `designNorthAngle` (user-set angle)
2. Apply it as a CSS transform to either:
   - `tnNeedleEl` (2D needle) with `rotate()`
   - `dnGroupEl` (3D group) with `rotateY()` for 3D perspective

The transforms are applied directly to DOM elements via `style.transform`.
